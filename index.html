Welcome to the Network Reliability Checker

This project helps in analyzing the reliability of a computer network by identifying cut vertices (articulation points) and biconnected components. These indicate critical nodes whose failure could break the network connectivity.

Navigate through the pages to learn how the algorithm works and try the interactive checker tool.

Project Overview

  • Uses Depth-First Search (DFS)-based Tarjan algorithm
  • Detects network points of failure
  • Runs completely in your browser (no backend needed)
